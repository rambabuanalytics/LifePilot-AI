# 🧠 LifePilot AI

LifePilot AI is an AI-powered Decision Intelligence Platform that helps users make smarter decisions by comparing two choices using Google's Gemini AI.

Instead of providing a simple answer, LifePilot AI generates a comprehensive decision report including comparison, risk analysis, confidence score, execution roadmap, and personalized recommendations.

---

## 🚀 Features

- 🤖 AI-powered decision analysis using Google Gemini AI
- ⚖️ Compare any two choices
- 📊 Detailed comparison table
- 🎯 Decision score with confidence level
- ⚠️ Risk analysis and hidden dependency detection
- 🗺️ Personalized execution plan
- 📅 Weekly roadmap
- 💡 AI coaching and productivity tips
- 📥 Downloadable AI-generated report
- 🎨 Modern Streamlit interface

## ⚙️ Installation

```bash
git clone https://github.com/your-username/LifePilot-AI.git

cd LifePilot-AI

pip install -r requirements.txt

streamlit run app.py
```

## 🔑 Environment Variables

Create a `.env` file and add your Gemini API Key.

```
GEMINI_API_KEY=YOUR_API_KEY
```
